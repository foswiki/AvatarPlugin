---+!! Avatar plugin

<img class="foswikiRight foswikiImage" src="%ATTACHURL%/example.jpg" />

%SHORTDESCRIPTION%

Add http://www.libravatar.org/ (or run your own libravatar server) or http://gravatar.com user images to your foswiki.

When this plugin is enabled, Any Links to user's home pages get that user's avatar image added to it.

%INCLUDE{VarAVATAR}%

---++ Configuration
You can select the avatar server to use in =Extensions= - =AvatarPlugin= - =AvatarServerBaseUrl= in =configure=
